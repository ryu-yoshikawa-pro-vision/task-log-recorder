# Code Review Entry Point

## 適用条件
- ユーザーがレビューを依頼した場合
- `/review` を使う場合
- 実装完了前の自己レビュー

## 使い方
1. `AGENTS.md` を確認する。
2. `.agents/skills/code-review/SKILL.md` を読む。
3. 必要に応じて `.agents/skills/code-review/references/review-workflow.md` を読む。

## 期待する出力
- findings-first
- severity 順
- 各 finding に根拠、影響、ファイル参照
- 問題がない場合も残余リスクと未実施検証を明記
